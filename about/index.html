项目简介

ReliToolbox 是面向过程/化工行业可靠性工程师的免费在线工具箱，旨在覆盖商业软件（ReliaSoft、Meridium、Maros、AvSim+）约80%的常用功能，帮助中小企业摆脱高昂的授权费用。所有计算在浏览器本地完成，无需服务器，数据完全私密。

功能模块详情（13个模块）
📉
Weibull 分析 / Weibull Analysis
寿命数据MLE拟合：Weibull / 指数 / 对数正态 / 正态四分布。B-Life特征寿命，AIC/BIC最优分布自动选择，支持右截尾数据，概率纸图，可靠度/危险率/CDF曲线，维修策略智能诊断（β<1早期故障/β≈1随机/β>1磨损）。高级模式：区间截尾+置信区间（Fisher信息矩阵）。
MLE · AIC/BIC · Fisher CI · Nelder-Mead · 截尾数据
📋
FMEA/RCM 工作表 / FMEA Worksheet
失效模式与影响分析：RPN自动计算（严重性×频率×可探测性），实时颜色预警（红≥200/橙≥100），Pareto图显示高风险失效模式，支持CSV导入/导出，IndexedDB本地持久化，多工作集管理，与设备台账联动。
FMEA · RPN · Pareto · IndexedDB · CSV
💰
LCC 生命周期成本 / Life Cycle Cost
多方案NPV净现值和EAC等效年成本对比，支持最多4个方案（继续维修/更换新设备/升级改造），折现率和分析期可调，累计成本曲线，自动识别最优方案。
NPV · EAC · 折现现金流 · 多方案对比
⚙️
RAM 仿真分析 / RAM Simulation
事件驱动蒙特卡洛仿真（最多5000次），k-out-of-n冗余结构，产能传递建模（串联系统容量=最小瓶颈），三级维修计划（设备级/装置级/全厂停车），指数/Weibull TTF，截断正态TTR，收敛曲线，Web Worker后台运行不阻塞UI。
事件驱动MC · k-of-n · 产能传递 · Web Worker · Weibull/Exp TTF
🛡️
RBI 基于风险检验 / Risk Based Inspection
基于API 580/581简化方法的5×5风险矩阵，失效可能性（PoF）和失效后果（CoF）分别1-5评分，剩余寿命计算（壁厚裕量/腐蚀速率），自动推荐检验间隔（1/3/5/10年），高风险设备优先排序。
API 580/581 · 5×5矩阵 · PoF/CoF · 剩余寿命 · 检验计划
🔧
备件库存优化 / Spare Parts Optimization
基于Poisson需求模型的备件库存优化，针对目标服务水平（80%-99.9%）计算最优库存量，关键性系数（A/B/C）调整，当前库存状态评估（OK/需审查/短缺），库存成本对比。
Poisson CDF · 服务水平优化 · 关键性权重 · 库存成本
🚨
SIL 安全完整性等级 / SIL Verification
基于IEC 61508/61511标准，计算安全仪表功能（SIF）的PFD（低要求模式）和PFH（高要求/连续模式），支持串联/并联架构，验证SIL等级（SIL1-4），考虑诊断覆盖率（DC）、共因失效（CCF β因子）、安全失效比例（SFF）。
IEC 61508/61511 · PFD · PFH · SIL1-4 · DC · CCF · SFF
📈
AMSAA 可靠性增长 / Reliability Growth
Crow-AMSAA（NHPP幂律过程）分析：MLE拟合β（增长参数）和λ，Duane图，累计失效曲线，当前MTBF估算，增长趋势判断（β<1改善中/β=1无变化/β>1退化），适用于研发测试和在役设备可靠性增长跟踪。
Crow-AMSAA · NHPP · MLE · Duane图 · 增长趋势
🌳
FTA 故障树分析 / Fault Tree Analysis
文本式门定义（AND/OR门），MOCUS算法求最小割集（MCS），Birnbaum重要度排序（识别最脆弱基本事件），顶事件概率精确计算（≤12割集容斥/>12稀有事件上界），模型保存至IndexedDB。
MOCUS · 最小割集 · Birnbaum重要度 · 容斥原理 · IndexedDB
📐
可用度分配 / Reliability Allocation
三种分配方法：AGREE法（考虑子系统复杂度、重要度、单元数），等分配法，加权因子法（按失效率反比权重）。自动验证∏Rᵢ≥R_sys，输出每子系统Rᵢ/MTBFᵢ/λᵢ。
AGREE · Equal · Weighted · 系统可靠度验证
🔗
RBD 最小割集 / RBD Minimal Cut Sets
接受RBD模型JSON，穷举法+超集剪枝求最小割集（支持最多18个节点），按阶数排序（1阶=单点失效=最危险），薄弱点分析（出现频率最高的节点），自动生成改进建议（单点失效警告、CCF提醒）。
最小割集 · 超集剪枝 · 单点失效 · CCF识别
⚙️
设备管理台账 / Equipment Registry
统一设备数据中台：CRUD完整台账管理（35个字段，涵盖基本信息/可靠性参数/工艺参数/检验计划/成本），KPI仪表盘（总设备/A类关键/检验到期/高风险），CSV双向导入导出，与FMEA/RBI/Weibull共享IndexedDB数据。
IndexedDB · CRUD · CSV双向 · KPI仪表盘 · 多模块联动
🖊️
RBD 拖拽画布 / RBD Visual Builder
可视化可靠性框图构建：拖拽节点，连接串/并联关系，导出JSON供MCS分析模块使用，支持门结构可视化编辑。
拖拽画布 · 串/并联 · JSON导出 · MCS联动
